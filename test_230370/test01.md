## Test 01: Consultar los registros por Estudiantes
---

#### Descripción:
Consulta SQL a la base de datos centralizada del proyecto de clase de e-commerce para la materia de base de datos para Negocios
Digitales

#### Objetivo:
Verificar que el estudiante:
- Comprende la estructura de base datos.
- Capaz de realizar una consulta SELECT correctamente
- Aplica filtros (*WHERE*) a la vista de **Registros Por Estudiante**


#### Criterios de Validación

-Muestra los registros que ha realizado la base de datos
-Deberá contar con 75 registros de personas
-Deberá contar con 50 registros de personas físicas
-Deberá contar con 25 registros de personas morales
-Deberá contar con 20 registros de productos
-Deberá contar con al menos 40 registros de categorización de productos (por aquellos que les toco una subcategoría)
-Deberá contar con 20 registros de categorización en la base de origen del producto (Nacional / Importación)
